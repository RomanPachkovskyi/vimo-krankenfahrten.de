# Studio Onepage Site

Minimal static React/Tailwind site used by the `_studio-onepage` generator.

## Scripts

```sh
npm install
npm run dev
npm run lint
npm test
npm run build
```

## Project Contract

- `src/config.ts` is generated by `tools/run-onepage.mjs`.
- The rest of the template is engine-managed and should stay generic.
- `npm run build` now emits a portable `dist/` that works for normal static hosting and direct `file://` opening.
- `npm run build:file` is kept as an alias for the same portable output.
